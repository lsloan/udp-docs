Learning Record Store queries

Unizin Common Data Model


Overview

The following document provides some example starter queries for the UDP Learning Record Store of commonly-modeled data. It is intended only to provide some starter queries from which Members may work as they explore data in the Unizin Common Data Model.

Example queries

Count events by type and action 

            SELECT
              COUNT(*) c, event_type, event_action
            FROM
              learning_datasets.enriched_events
            GROUP BY
             event_type, event_action
            ORDER BY c DESC
            ;

Find all of the "grade" events for a particular user 

            SELECT
              event
            FROM
              learning_datasets.enriched_events
            WHERE
              event_type = 'GradeEvent' AND
              ucdm_actor_id = '1'
